<h1 align="center">
    sdl-wallpaper
</h1>

> Example code to put an SDL window as the wallpaper on Windows

## Requirements
- [CMake](https://cmake.org/download/)
- [SDL2](https://www.libsdl.org/download-2.0.php)

## Install
```
git clone https://github.com/TrAyZeN/sdl-wallpaper.git
cd sdl-wallpaper
cmake .
```
